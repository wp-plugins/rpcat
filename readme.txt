== Description ==

This plugin creates a widget to allow you to show recent posts in your sidebar from a specific category.

Features:

* You can add this widget as many times as you like on your blog.
* You can customize the header of each one of the widget instances.
* You can choose how many items to show with a maximum of 15.

== Installation ==

1. Upload `rpcat` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. You can now use it from the widgets menu.
